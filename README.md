# auto-feedbackform-filler
Bookmarks which automatically fill out the feedback form with correct options

just save the bookmark as per the option

HOW TO USE

**1) Copy the code in the respective txt file, then go to your browser bookmarks bar. Right-click → Add new bookmark.**

**2) Set any name (e.g. "Fill Feedback"), then paste the code into the URL / Address field.**

**3) Navigate to your feedback page, then click the bookmark. All dropdowns will be filled instantly.**
